Simple Steel

My first official Minecraft mod. It's aim? To supply a lightweight mod to add steel.

Project Date: July 2018 - January 2021

Description

This was my first attempt to write a Minecraft mod, the idea was to serve as a root for other mods/packs to easily integrate basic steel production and tools into their games.
I stopped working on this after 1.12, with an eventual release for 1.16 afterwards I decided the idea was not needed because other larger mods having steel included.
I did learn a number of skills from making this mod, both learning Java and a lot of concepts such as interfaces and events.
