Game Jam 2020

*prototype zombie noises*

Project Date: September 2020

Description

This project is one of the few projects where I didn’t write any code, I oversaw the 3D assets and level design, Lewis Witts worked on material creation, sound design and additional programming and Sean was our project lead and main programmer.
We had 24 hours to create this game and we spent the full 24 hours; I learnt a lot about working with others and doing different roles.
The idea for the game was to be a wizard surviving the zombie apocalypse after he caused the end of the world with a … fireball of course.
